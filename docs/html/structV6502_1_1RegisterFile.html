Detailed Description

Register File for the 6502 processor.

The register file for the 6502 is accumulator based. The registers indexX and indexY are used for calculating offsets for some of the many addressing modes available to the 6502.

The program counter is 16 bit allowing the processor to acess up to 64 KB of RAM. The stack pointer is only 8 bit, although when it is used in reading or writing to memory, it is extended to 16 bits, with the upper word being set to the value of 1.
